# An Essay on House Cleaning

> In this note we will explore a set of related problems cleaning the house.

## Problem Set

- Surface cleaning: floors, tables, counters.
- Air cleaning: ventilation, odor control, humidity regulation.
- Object cleaning: dishes, clothes, furniture.
- Waste management: removal, classification, recycling.

## The Scheduling Problem

How to distribute cleaning activities in time to minimize disorder and fatigue.

- Daily: tidying, dishes, surfaces.
- Weekly: floors, laundry, bathroom, dusting.
- Monthly: deep cleaning, fridge, curtains, windows.
