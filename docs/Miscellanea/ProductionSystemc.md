## Structure

| Layer | Includes |
| --- | --- |
| **Physical** | Machines, layout, raw materials, workers |
| **Information** | Orders, instructions, quality feedback |
| **Control** | Scheduling, flow control, automation, feedback |
| **Cognitive** | Process planning, improvement, system learning |
| **Cultural** | Kaizen, TPM, worker empowerment, norms |

## Framework

| Framework | Why It Matters |
| --- | --- |
| **Toyota Production System (TPS)** | Foundation for lean, flow, just-in-time, and kaizen. |
| **Theory of Constraints (TOC)** | Teaches systemic thinking and bottleneck optimization. |
| **Lean Manufacturing** | Eliminates waste, improves flow, empowers people. |
| **Industrial Engineering / Work Study** | Time-motion studies, ergonomics, efficiency. |
| **Systems Engineering** | Helps design and integrate the whole production system. |

## Related Terms

| Name | Scope | When to Use |
| --- | --- | --- |
| **Manufacturing** | Physical act of making | Narrow or task-level view |
| **Production System** | Structure of how manufacturing is organized | Systems-level view (recommended) |
| **Operations System** | Broader: includes service and logistics | Business-level view |
| **Value Creation System** | Includes design, manufacturing, delivery | Strategic-level thinking |
